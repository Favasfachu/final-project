## Overview
This project focuses on performing exploratory data analysis (EDA) on a car dataset using Python.  
The objective is to understand the structure of the data, identify patterns, relationships, and trends, and visualize insights using graphical analysis.

---

## Objectives
- Perform data cleaning and preprocessing
- Conduct univariate and bivariate analysis
- Analyze relationships between key variables
- Visualize data insights using charts and plots

---

## Dataset
The dataset contains information related to cars such as:
- Brand / Model
- Year
- Price
- Mileage
- Fuel type
- Transmission
- Engine specifications

*(Dataset used for educational and analytical purposes)*

---

## Analysis Performed
- Data inspection and cleaning
- Handling missing values
- Univariate analysis
- Bivariate analysis
- Correlation analysis
- Graphical analysis using visualizations

---

- Python
- Pandas

- Matplotlib
- Seaborn
